# ChessEngineCPP

> A chess game desgined in the C++ programming language
# Init
The code starts by initializing the board, using stringified plotting, going each character and plotting the right piece on the board for it, the basic board string is `rnbqkbnrppppppppddddPPPPPPPPRNBQKBNR` which puts a Rook (`r`), a Knight (`n`), a Bishop (`b`), a Queen (`q`), a King (`k`), the player's pawns, goes down 4 rows (`dddd`), and puts the others for the other player.
